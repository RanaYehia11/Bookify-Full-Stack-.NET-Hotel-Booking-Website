**Full Stack .NET Web Application | ASP.NET Core MVC | N-Tier Architecture**

Bookify is a full-featured hotel reservation web application built using **ASP.NET Core MVC**, following clean architectural principles like **N-Tier Architecture**, **Repository Pattern**, and **Unit of Work Pattern**.  
The system provides room browsing, availability checking, reservation management, Stripe online payments, and a full Admin Panel for hotel staff.

---

##  Features

###  **Customer Features**
- View room types and available rooms  
- Search by availability dates  
- Room details page  
- Add rooms to reservation cart (Session-based)  
- Secure Stripe payment integration  
- Booking confirmation  
- Personal profile with booking history  
- User authentication (Register/Login)

###  **Admin Panel**
- Role-based authorization (Admin / Customer)  
- Manage Room Types  
- Manage Rooms  
- Manage Bookings  
- DataTables integration for advanced table UI  
- Dashboard overview

###  **Core Architectural Features**
- N-Tier Architecture (Web, Services, Data)  
- Repository Pattern for clean data access  
- Unit of Work handling transactional operations  
- Dependency Injection  
- ASP.NET Identity for authentication  
- Entity Framework Core (Code-First Migrations)  
- Serilog Structured Logging  
- ASP.NET Core Health Checks (`/health` endpoint)
